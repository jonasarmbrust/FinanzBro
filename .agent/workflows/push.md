// turbo-all

1. Stage all changes:

```powershell
git add -A
```

2. Check what will be committed:

```powershell
git status
```

3. Create a descriptive commit message based on the changes made. Use German for the commit message if the changes were discussed in German:

```powershell
git commit -m "<beschreibende Nachricht>"
```

4. Push to the remote repository:

```powershell
git push
```
